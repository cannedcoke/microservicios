microservicios
==============

Implementacion de una arquitectura de microservicios desarrollada en Python con Flask.

Descripcion
-----------
Proyecto que modela una aplicacion basada en microservicios, donde cada servicio es independiente, tiene su propia responsabilidad y se comunica con los demas a traves de una interfaz definida. El archivo app.py actua como punto de entrada principal y el directorio services contiene cada microservicio de forma separada.

Tecnologias
-----------
- Python
- Flask

Estructura
----------
- app.py            Punto de entrada y coordinacion de servicios
- services/         Directorio con cada microservicio independiente
- requirements.txt  Dependencias del proyecto

Como ejecutar
-------------
1. Instalar dependencias: pip install -r requirements.txt
2. Ejecutar la aplicacion: python app.py
